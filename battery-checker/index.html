How to generate Android dumpstate?

  1. Open phone dialer, type *#9900# (Samsung devices).
  2. Tap "Run dumpstate/logcat".
  3. Select "Copy to SD card" or share/transfer the resulting .log file to your computer.
  4. Alternatively, run adb bugreport via computer terminal.
