# web-apps

A few online utilities. Built with [Cursor](https://www.cursor.com), [Claude](https://claude.ai) and [Gemini](https://gemini.google.com).

| App                                                                                     | Description                                                                                   |
| --------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| [2FA Authenticator](https://vietanhbui2000.github.io/web-apps/2fa-auth/)                | Simple, secure 2FA authenticator that works locally in the browser.                           |
| [Blank Page](https://vietanhbui2000.github.io/web-apps/blank-page/)                     | An infinite blank page to write everything in the browser.                                    |
| [Clock Widget Builder](https://vietanhbui2000.github.io/web-apps/clock-widget-builder/) | Create and customize a clock widget with live preview and get embed code for Obsidian/Notion. |
